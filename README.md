# Selenium Login Automation Project

## 📌 Project Overview
This project automates the login functionality of a web application using Selenium WebDriver and TestNG framework.

The objective of this project is to validate login functionality with both valid and invalid credentials and ensure application stability.

---

## 🛠 Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- GitHub

---

## 📂 Project Structure
Selenium-Login-Automation
 ├── src/test/java
 │    └── LoginTest.java
 ├── testng.xml
 ├── pom.xml
 └── README.md

---

## 🚀 Features Automated
✔ Login with valid credentials  
✔ Login with invalid credentials  
✔ Error message validation  
✔ Browser launch and close  

---

## ▶ How to Execute
1. Install Java and Maven
2. Import project into Eclipse or IntelliJ
3. Update WebDriver path
4. Run testng.xml file
